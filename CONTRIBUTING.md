# Contributing Guidelines

Thanks for your interest in contributing to the PSTU Bioinfo Club website!

## How to Contribute

- Fork the repo and create a feature branch.
- Keep changes focused and scoped.
- Follow the existing code style and patterns.
- Update or add documentation where relevant.

## Setup

```zsh
bun install
bun run dev
# or
npm install
npm run dev
```

## Code Style

- TypeScript strictness preferred.
- Use Tailwind utilities; avoid custom CSS unless necessary.
- Keep components small and reusable.

## Commit Messages

Use clear, imperative messages, e.g., `feat: add events page`.

## Pull Requests

- Describe the change, motivation, and testing notes.
- Link any related issues.
- Ensure `bun run build` (or `npm run build`) succeeds locally.

## Issues

- Provide steps to reproduce, expected vs actual behavior, and environment details.
