# PSTU Bioinfo Club

A Vite + React + TypeScript + Tailwind project for the PSTU Bioinformatics Club website.

## Tech Stack

- Vite
- React (SWC)
- TypeScript
- Tailwind CSS

## Getting Started

### Prerequisites

- Node.js 18+
- npm or bun

### Install

```zsh
# using bun
bun install

# or using npm
npm install
```

### Development

```zsh
# start dev server
bun run dev
# or
npm run dev
```

### Build

```zsh
bun run build
# or
npm run build
```

### Preview

```zsh
bun run preview
# or
npm run preview
```
